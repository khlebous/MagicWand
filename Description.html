Magic Wand

The aim is to implement application with floating rectangles. Not described issues should imitate example application

Basic functionality - part I

  • There are 20 rectangles floating on a screen appearing every 100 ms. Their start position is always the same position as a mouse cursor, however they have different velocity and colors. Every time application is opened, all variables are different.
  • After 5 seconds, each window disapears.
  • All rectangles are top - most
  • All rectangles bounces from the screen - they should never leave it
  • Pressing ALT-F4 should close last created rectangle
  • All rectangles should be windows without top controls. They should also not be visible on a menu
  • There are no tips, as they varies on technology used.
  • Scoring:
    • Generate 20 rectangles in the mouse position with different colors. Pressing ALT-F4 should close last window: 4 points
    • Add movement, bounce and 5 seconds timeout, after which rectangle disapears - 4 points

Extended functionality - part II

  • We will be changing rectangle windows to either a circular windows or an equilateral polygon (3-7 edges)
  • Opacity is constantly changing from 100% to 0% and then back to 100% - step varies depending on a window
  • In case of windows in a shape of equilateral polygons, they are constantly rotating
  • Scoring:
    • Change windows shapes from rectangle to circle or polygon - 4 points
    • Add constantly changing opacity - 4 points
    • Add rotation - 4 points
